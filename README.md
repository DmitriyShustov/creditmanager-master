# CreditManager

This project represents the basic functionality of the credit manager's automated workplace.

## Deploying

For deploying you need Docker on your computer

### Run the following commands in your terminal:

1. git clone https://github.com/DmitriyShustov/creditmanager-master.git
2. cd creditmanager-master
3. git submodule update --init --recursive
4. docker compose up --build
